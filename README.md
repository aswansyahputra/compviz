# compviz

[![Launch Rstudio Binder](http://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/aswansyahputra/compviz/master?urlpath=rstudio) 

This repository stores all materials for Dipo Analytics webinar about data visualization. You can play with `demo.R` to build and compose graphs. Please consult to `data-raw` if you want to learn the data preprocessing steps.

You can run a virtual RStudio by clicking the badge above. Happy learning!
